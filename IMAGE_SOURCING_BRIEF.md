## Source Recommendations

**Free Stock Photo Sites:**
1. **Unsplash** (unsplash.com) — High quality, modern aesthetic
2. **Pexels** (pexels.com) — Good for lifestyle and business
3. **Pixabay** (pixabay.com) — Large library, varying quality

**Premium Stock (if budget allows):**
1. **Shutterstock** — Best selection for Filipino/Asian-specific content
2. **Adobe Stock** — High quality, integrated with design tools
3. **iStock** — Good for authentic lifestyle imagery

**Search Tips:**
- Add "Philippines" or "Filipino" to searches for local content
- Use "Southeast Asia" or "Asian" as fallback
- Search for specific settings: "Manila", "tropical home", "Asian neighborhood"
- Look for images with natural lighting and genuine expressions

## File Naming Convention

When downloading, rename files using this convention:

```
section-purpose.jpg

Examples:
- about-community.jpg
- how-it-works-consultation.jpg
- how-it-works-savings.jpg
- how-it-works-eligibility.jpg
- solutions-cost-savings.jpg
- solutions-dashboard.jpg
- whyus-solar-rooftop.jpg
- testimonials-partnership.jpg
```

## Image Specifications

| Property | Requirement |
|----------|-------------|
| Format | JPG or WebP |
| Color Space | sRGB |
| Resolution | Minimum 1200px wide |
| Aspect Ratios | 16:9, 4:3, or 3:2 |
| File Size | Under 500KB optimized |
| Quality | No watermarks, high resolution |

## Delivery Instructions

1. Download images to: `/public/images/sections/` in the project
2. Use the naming convention above
3. Optimize images using tools like TinyPNG or Squoosh
4. Create a brief note for each image describing:
   - Source (Unsplash, Pexels, etc.)
   - Photographer credit (for attribution)
   - License type
